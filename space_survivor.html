Space Survivor

2D GAME

Space Survivor is a 2D game built with python using the Arcade Library.

The history behind the game:

The story behind the game "space survivor save the galaxy" is a story of a hero that stands against an alien invasion, with his spaceship he tries to defender the earth and eliminate the enemy's spaceships.

The Arcade Library:

The arcade library is a modern Python framework for crafting games with compelling graphics and sound. Object-oriented and built for Python 3.6 and up, arcade provides the programmer with a modern set of tools for crafting great Python game experiences.

Technologies:

In addition to the arcade library that allows building the game engine and logic, I used also the cross-platform 2D-3D vector graphics OpenGL and the cross-platform 3D audio OpenAL in order to add graphics and sounds inside the game and enhance the game experience and add reaction such as: the explosion effect and bullets sound.

Installing and playing Space-Survivor:

Installation:

In order to play the game you have first to install the Arcade Library using the source code and the ideal way is to pick up the how to please refer to this link.

If you run into issues or error, please verify the right installation of dependencies, create an issue on the arcade library repo, or reach me by email.
