Smiling School

A full dynamic website

The Smiling school is a static website built from scratch as a front-end curriculum final project. It summarizes all that I learned in front-end development: responsive design, devtools, Figma design, Accessibility design.

Programming Languages and Technologies:

From Figma layouts to an animate website, I used HTML, CSS, Bootstrap, JavaScript, and JQuery to create this project.

During the project development, I used devtools in order to debug styling issues and layouts behaviors with different screen sizes.

Styles and animations:

The website support responsiveness on desktop, iPad, and mobile screens.

The website's pages are built using different CSS layouts to display the content in 2-dimensional using "Grid" and one-dimensional using "Flexbox".

In order to dispaly multi-content I used bootstrap responsive carousel animated dynamically with Javascript and JQuery.

Implement a CSS animation and Javascript scripts to create a loader when requesting new data.

Manipulate the DOM (Document Object Model), by impelementing cards, quotes, loader in page dynamically.
